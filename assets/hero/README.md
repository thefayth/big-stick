# Hero Assets

Public-safe hero assets for the Big Stick repository and FaithCheltenham.com
draft page.

- `hero-image.svg` - source hero artwork derived from the Big Stick public
  featured image.
- `hero-image.png` - rendered PNG hero image for README, WordPress, and social
  systems that prefer raster assets.

No private screenshots, route captures, records, or protected dashboard imagery
belong in this folder.
