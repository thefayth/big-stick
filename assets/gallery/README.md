# Visual Gallery

Public-safe visual assets selected or prepared for the Big Stick public
surface.

## Included

- `big-stick-logo-mark.svg` - project icon and compact identity mark.
- `big-stick-hero-panel.svg` - hero image from existing Big Stick brand assets.
- `big-stick-wordpress-featured.svg` - prepared WordPress featured image.

## Not Included

Private screenshots, raw records, local route captures, receipts, transcripts,
and operational UI captures are excluded from this gallery.
