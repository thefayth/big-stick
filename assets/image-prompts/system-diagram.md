# System Diagram Prompt

## Filename Suggestion

`big-stick-public-private-boundary-diagram.svg`

## Purpose

Explain public/private separation without exposing implementation details.

## Placement

WordPress section art and public docs.

## Aspect Ratio

4:3 or wide 16:9.

## Diagram Content

Show four abstract zones:

1. Public Record.
2. Protected Dashboard.
3. Private Vault And Transcripts.
4. Review And Export.

Use arrows that show reviewed summaries moving outward and raw/private material
staying inside.

## Avoid

No real route names beyond high-level labels. No storage paths. No source code.
No API names. No credentials. No private records.

## Accessibility Alt Text

Diagram showing reviewed public summaries moving outward while private records
remain protected inside Big Stick.

## Privacy Notes

Can be generated as a simple abstract diagram after Faith review.
