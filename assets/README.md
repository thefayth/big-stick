# Assets

This folder contains public-safe image prompts, selected visuals, rendered PNGs,
workflow diagrams, hero assets, social assets, banner assets, gallery assets,
and icon assets.

No private screenshots, records, transcripts, receipts, legal/admin/medical
documents, credentials, case details, names, emails, phone numbers, addresses,
or operational app screens belong here.

All visual assets should be reviewed by Faith before publication.

## Folders

- `banners/` - GitHub banner SVG and PNG.
- `hero/` - README and WordPress hero image.
- `social/` - social preview card SVG and PNG.
- `gallery/` - public-safe visual gallery assets.
- `icons/` - project icon assets.
- `diagrams/` - workflow and public/private boundary visuals.
- `image-prompts/` - future image generation and Canva direction.
