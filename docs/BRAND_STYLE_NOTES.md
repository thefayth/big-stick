# Brand Style Notes

## Public Identity

Big Stick should read as protective, serious, operational, and owned. It is not
a generic SaaS dashboard and should not look like a public evidence dump.

## Visual Language

- Deep ink backgrounds.
- Warm paper panels.
- Graphite structure lines.
- Restrained red redaction marks.
- Muted gold receipt tabs.
- Clear public/private boundary motifs.
- Command-board density without visual panic.

## Voice

- Clear.
- Protective.
- Direct.
- Calm under pressure.
- Public-safe.
- No hype, no sales funnel, no "download now" energy.

## Avoid

- Generic blue SaaS gradients.
- Courtroom cliches such as gavels and scales.
- Surveillance imagery.
- Real private screenshots.
- Public app availability implications.
- Militancy or violent visual metaphors.
- Open-source or free-use implications.
