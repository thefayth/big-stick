# Project Brief: Big Stick

## One-Sentence Description

Big Stick is a protected local command center for managing public-record work,
private evidence review, archive retrieval, listening/transcript intake,
AI-assisted drafts, and export readiness.

## What It Is

Inspected files describe Big Stick as a local Next.js desktop-capable app with
one primary brand and visible tool areas: Command, Fightback: Public Record,
Unburied: Vault, Listen Up, AI Console, Exports, and Settings.

It separates public-facing Fightback pages from protected dashboard routes and
keeps raw/private material behind review and redaction boundaries.

## What It Is Not

- Not a public download.
- Not an open-source release.
- Not a legal-advice product.
- Not a public evidence dump.
- Not a sales page.
- Not a replacement for human review, legal judgment, or publication approval.

## Who It May Eventually Serve

Unknown from inspected files as a public offering. The inspected project is
currently Faith's protected local command system. Possible future audiences
could include people or teams doing careful evidence, public-record, advocacy,
archive, or documentation work, but no public pilot is offered here.

## Why It Matters

The project treats privacy, provenance, redaction, review, receipts, and
public/private separation as core product features rather than afterthoughts.
